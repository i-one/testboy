testboy.asserts

Provides assertion methods. Only failed assertions are recorded. These methods can be used directly: testboy.asserts.assertTrue(...), however they are available in the global context: assertTrue(...).

Methods

assertEquals
void assertEquals(expected, actual, message)

Asserts that two values are equal.

param boolean expected Expected value.
param boolean actual Value to be checked against expected.
param string message Optional. Message for the AssertionError to be displayed if the assertion fails.
assertError
void assertError(fun, args, thisArg, message)

Asserts that a function throws an error.

param Function fun Function to test.
param Object args The arguments object or an array of arguments to pass in fun.
param Object thisArg Determines the value of this inside fun. If thisArg is null or undefined, this will be the global object. Otherwise, this will be equal to Object(thisArg).
param string message Optional. Message for the AssertionError to be displayed if the assertion fails.
